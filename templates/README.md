## Template Customization Guide

### For Different Domains

**Climate Change** (default):
- Use H‚ÇÉ (trust) as typical bottleneck
- Reference Paris Agreement, COP processes
- Emphasize long-term cooperation needs

**Pandemic Preparedness**:
- Focus on H‚ÇÅ (governance) and H‚ÇÉ (trust)
- Reference COVID-19 coordination failures
- Emphasize rapid response coordination

**AI Governance**:
- Highlight H‚ÇÅ (institutions) and H‚Çá (technology) imbalance
